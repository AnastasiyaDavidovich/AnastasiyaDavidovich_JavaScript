Дана скобочная последовательность: [((())()(())]]
- Можно ли считать эту последовательность правильной?
- Если ответ на предыдущий вопрос “нет” - то что необходимо в ней изменить, чтоб она стала правильной?

Данная посленовательность не является правильной, т.к. не соблюдается условие, по которому последовательность должна обеспечивать последовательную вложенность подпоследовательностей, обрамленных открытой и закрытой скобкой одного типа.
Т.е. все скобки данной последовательности можно разбить на пары, для которых:
- в каждой паре есть открывающая и закрывающая скобки, причем открывающая расположена левее закрывающей;
- для любых двух пар скобок либо одна из них полностью находится внутри другой пары, либо промежутки между скобками в парах не пересекаются;
- в паре с круглой скобкой может быть только круглая, с квадратной - квадратная, с фигурной - фигурная.

Проверить является ли скобочная последовательность правильной можно с использованием функции на JavaScript:

function isBrakets(str){
   let arr = str.split('');
    for (let i = 0;i<arr.length;i++){
      if (arr[i]=='(' && arr[i+1] == ')'|| arr[i]=='[' && arr[i+1] == ']'){
        arr.splice(i,2);
        i-=2;
      }
    }
   return arr.length == 0
  }




Сделать данную последовательность правильной можно несколькими способами:
1. [[((()))()(())]] - добавить в начале открывающую квадратную скобку и после первого блока открывающих круглых скобок добавить закрывающую круглую скобку.
2. [(())()(())] - удалить "лишние" скобки: закрывающую квардратную в конце последовательности и открывающую круглую в начале.
3. [()(())()(())[]] - добавить закрывающую круглую после первой открывающей круглой и открывающую квадратную перед первой закрывающей квадратной.
и т.д.
